lightworks-arch
===============

Lightworks arch PKGBUILD

How to
======

Clone this repo
  
    $ git clone git://github.com/bhush9/lightworks-arch.git

Change directory to lightworks
  
    $ cd lightworks
    
Make package using makepkg command.

    $ makepkg
    
Install newly created package.

    $ sudo pacman -U *.pkg.tar.xz
